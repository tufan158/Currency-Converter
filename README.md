# Currency-Converter

A simple and efficient web application that converts currencies in real-time using live exchange rates. This is my second project focusing on API integration and Object-Oriented JavaScript.

 Features
Live Rates: Fetches data from FreeCurrencyAPI.

Instant Results: Converts automatically as you type.

Modern JS: Built using ES6 Classes and async/await.

Clean UI: Simple, responsive design with FontAwesome icons.

 Technologies
HTML5, CSS3, JavaScript (ES6)

FreeCurrencyAPI

 Quick Start
Clone the repo.

Open index.html in your browser.

Enter the amount and select your currencies to see the result.
